
## 整体布局逻辑

按"保护链"分 **6 大板块**，覆盖从硬件到应用的全链路：

```
硬件层 (4项) → 数据层 (4项) → 协同层 (4项) → AI层 (4项) → 系统层 (2项) → 应用层 (2项)
```

**布局原则：**

- ✅ **近中远搭配**：Phase 0 就能申请的（硬件/结构）→ Phase 1-2 才成熟的（算法/AI）
- ✅ **对外展示价值**：融资时能拿出来讲的故事
- ✅ **竞品封锁**：卡住 Chemspeed/奔曜/晶泰 没做且难复制的方向

## 板块一：触觉传感硬件（4项）— Phase 0 可申请，最硬核壁垒

### 专利 ①：高密度织物压阻传感阵列结构

- **技术点**：三层织物叠层结构 + 锯齿形正交走线
- **创新点**：刺绣锯齿形电极走线实现 3-4mm 极小传感点，全柔性透气
- **对标**：PPS 65 点只能做到~10mm pitch，我们 3mm
- **对外展示**：⭐️⭐️⭐️⭐️⭐️ 实物拿得出手，投资人看得懂
- **优先级**：🚀 **第一批，立即启动**

### 专利 ②：零电位法串扰抑制电路与扫描方法

- **技术点**：矩阵扫描阵列的串扰消除电路，零电位接地 + 高速 ADC 逐行扫描
- **创新点**：适配织物大面阵的专用电路架构（区别于标准 FSR 薄膜方案）
- **对标**：参考 BYU RAD Lab 但做工程化改良
- **对外展示**：内行才看得懂，但对壁垒构建重要
- **优先级**：📌 Phase 0，随硬件同步申报

### 专利 ③：织物压阻传感阵列批次标定与补偿方法

- **技术点**：自动标定流程 + 可调增益采集电路 + 迟滞补偿算法（Prandtl-Ishlinskii 模型）
- **创新点**：解决织物传感器量产一致性问题，这是工程化落地的核心know-how
- **对外展示**：⭐️⭐️⭐️ 体现工程落地能力
- **优先级**：📌 Phase 1，有一定数据积累后申请

### 专利 ④：多模态融合数据手套

- **技术点**：织物触觉阵列 + 多指 IMU + 视觉标记的一体化穿戴设备
- **创新点**：触觉/惯性/视觉三模态在同一手套上集成，124+ 传感点
- **对标**：Manus/5DT 只有姿态追踪无触觉，PPS 只有触觉无IMU
- **对外展示**：⭐️⭐️⭐️⭐️⭐️ **最强对外展示专利，全球独一份**
- **优先级**：🚀 **第一批，与专利①作为组合拳**

## 板块二：多模态数据采集与融合（4项）— Phase 1 可申请，数据护城河

### 专利 ⑤：多源异构传感器实时同步方法

- **技术点**：触觉(15Hz) + IMU(100Hz) + 视觉(30fps) 三路数据的高精度时间戳对齐
- **创新点**：异帧率传感器在化学实验慢速操作场景下的软硬件同步方案
- **对外展示**：⭐️⭐️ 偏底层，但对数据质量至关重要
- **优先级**：📌 Phase 1

### 专利 ⑥：触觉压力分布到手指关节角度的映射方法

- **技术点**：124+传感点压力分布 → 5指15关节角度的非线性映射算法
- **创新点**：最大滤波 + 归一化 + 线性/非线性映射 + 全开/全握自动标定
- **对外展示**：⭐️⭐️⭐️⭐️ 体现"触觉→运动"的核心算法能力
- **优先级**：📌 Phase 1，手套数据稳定后申请

### 专利 ⑦：基于触觉手套的化学实验操作动作分类与编码方法

- **技术点**：抓取/倾倒/搅拌/旋紧/刮涂等化学操作的标准触觉特征编码
- **创新点**：建立化学实验操作的标准触觉数据库和分类体系
- **对外展示**：⭐️⭐️⭐️⭐️ **融资时讲"标准化数据资产"的故事**
- **优先级**：📌 Phase 1-2，有一定数据积累后

### 专利 ⑧：化学实验操作数据质量评估与清洗方法

- **技术点**：自动识别异常操作/传感器失效/操作不规范的数据质量流水线
- **创新点**：针对化学实验场景的专用数据清洗（非通用数据清洗）
- **对外展示**：⭐️⭐️ 体现工程成熟度
- **优先级**：📌 Phase 1 后期

## 板块三：人机协同与遥操作（4项）— Phase 1-2 可申请，核心壁垒

### 专利 ⑨：基于触觉手套的人手到机械臂操作映射方法

- **技术点**：人手27自由度 → 6自由度机械臂的运动重映射与轨迹转换
- **创新点**：解决 Correspondence Problem 的专用映射算法（含 DTW 轨迹对齐 + 速度缩放）
- **对外展示**：⭐️⭐️⭐️⭐️ **体现"人教机器人"的故事线**
- **优先级**：🚀 **第一批，与硬件同步**

### 专利 ⑩：触觉反馈辅助的遥操作化学实验系统

- **技术点**：操作者在安全距离外通过触觉手套遥控机械臂，同时接收力反馈
- **创新点**：解决有毒有害化学环境下的"人机安全分离"操作
- **对外展示**：⭐️⭐️⭐️⭐️⭐️ **融资演示最佳场景，视觉冲击力强**
- **优先级**：📌 Phase 1-2

### 专利 ⑪：基于触觉感知的机器人主动辅助协同方法

- **技术点**：机器人通过观察人手触觉数据预判下一步动作并主动辅助
- **创新点**：触觉先于视觉感知操作意图（视觉看到手在动时已晚，触觉感知发力意图更早）
- **对外展示**：⭐️⭐️⭐️⭐️⭐️ **颠覆性创新，全球未见同类专利**
- **优先级**：🚀 **与其他专利错开，Phase 1 中期启动，保密到Demo成熟**

### 专利 ⑫：人机安全交互的碰撞检测与柔顺控制方法

- **技术点**：工作空间内人与机器人的碰撞检测 + 紧急停止 + 柔顺控制
- **创新点**：结合触觉手套数据辅助的安全策略（人戴手套时，机器人知道人的手在哪）
- **对外展示**：⭐️⭐️⭐️ 安全主题，给投资人/政府看更靠谱
- **优先级**：📌 Phase 2
