# 原位 DRIFTS 双臂灵巧手机器人操作流程

> **来源**：浙大研究生整理
> **文件**：[[原位红外双臂灵巧手机器人操作要求.pdf]]
> **录入**：2026-07-23

## 概述

自动化原位红外（DRIFTS）实验的全流程操作规范，以 Ø4.7mm 微量杯为边界条件，覆盖从配方、加样到清洁的完整 24 步工艺链。

## 核心结论

**±0.1mm 机械臂重复精度可用**，但必须叠加：
- 近景视觉（建议专用相机 ≤0.026mm/px）
- 导向夹具
- 力控闭环（0.05-0.10N 触碰精度）

## 24 步全流程

### A — 配方与追溯（#1-4）
识别样品/杯/池 → 载入 SOP 与风险 → 生成实验任务

### B — 制样与装杯（#4-6）
粉末取样 → 粗加样/微加样 → **抹平与表面质检** ⭐

> 抹平是关键工艺：粉末表面影响散射系数 → 直接影响光谱重复性

### C — 装池与密封（#7-9）
转移微量杯 → 杯坐底确认 → O形圈/窗片/上盖

### D — 接气与装机（#10-12）
接气/热电偶/加热 → 检漏 → 原位池装入光谱仪

### E — 光路与背景（#13-15）
杯位/高度调整 → 吹扫与信号稳定 → 背景/初始谱

### F — 原位实验（#16-19）
预处理 → 气体/温度程序 → 事件触发采谱 → 在线光谱质控

### G — 停机与清洁（#20-24）
惰性吹扫/冷却 → 卸压断开 → 取池取杯 → 清洁检查 → 数据记录

## 精度要求

| 参数 | POC 阶段 | 成熟阶段 |
|------|---------|---------|
| 工具-杯中心 | ±0.25mm | ±0.15mm |
| 粉面高度 | ±0.15mm | ±0.10mm |
| 粉面峰谷差 | ≤0.10mm | ≤0.10mm |
| 触碰力 | 0.05-0.10N | 0.05-0.10N |
| 抹平力 | 0.1-0.5N | 0.1-0.5N |

## 双臂分工

- **左臂**：稳定杯座/原位池/附件
- **右臂**：取工具、加样、抹平、紧固、接驳
