# 原位 DRIFTS 自动化 — LabVLA 适配性评估

> **日期**：2026-07-23
> **来源**：浙大研究生整理的《原位红外双臂灵巧手机器人操作要求》
> **评估**：传感前锋 ⚡

---

## 综述

本文档系统评估 LabVLA 框架是否能完成原位 DRIFTS（漫反射红外傅里叶变换光谱）实验的全流程自动化，以及需要补充的技术能力。

---

## 一、DRIFTS 实验全景

### 实验原理

原位 DRIFTS 是催化领域最常用的表征手段之一——一边让催化剂"工作"（升温+通气），一边用红外光实时观察催化剂表面发生了什么变化。

### 24 步全流程

| 阶段 | 编号 | 操作内容 | 粒度 |
|------|------|---------|------|
| **A 配方追溯** | 1-4 | 识别样品/杯/池 → 载入SOP → 生成任务 | 高层认知 |
| **B 制样装杯** | 4-6 | 粉末取样 → 粗加样/微加样 → **抹平与表面质检** ⭐ | **精密操作** |
| **C 装池密封** | 7-9 | 转移微量杯 → 坐底确认 → O形圈/窗片/上盖 | 精密装配 |
| **D 接气装机** | 10-12 | 接气/热电偶/加热 → 检漏 → 入光谱仪 | 粗操作 |
| **E 光路背景** | 13-15 | 杯位调整 → 吹扫稳定 → 背景谱 | 光学对准 |
| **F 原位实验** | 16-19 | 预处理 → 温程 → 触发采谱 → 在线质控 | 流程控制 |
| **G 停机清洁** | 20-24 | 惰吹扫 → 卸压 → 取池取杯 → 清洁记录 | 粗操作 |

### 核心精度参数

| 参数 | POC 阶段 | 成熟阶段 |
|------|---------|---------|
| 工具-杯中心 | ±0.25mm | ±0.15mm |
| 粉面高度 | ±0.15mm | ±0.10mm |
| 粉面峰谷差 | ≤0.10mm | ≤0.10mm |
| 触碰力 | 0.05-0.10N | 0.05-0.10N |
| 抹平力 | 0.1-0.5N | 0.1-0.5N |

### 边界条件

- 样品杯：Ø4.7mm，深 1.6-4mm，体积约 30µL
- 工具套件：微勺、漏斗、刮片、O形圈、热电偶、气路接头
- 环境要求：远离热区、粉尘、气体泄漏

## 二、LabVLA 适配性评估

### 当前 LabVLA 状态

| 项目 | 状态 |
|------|------|
| 模型 | LabVLA-5B-Base（Qwen3-VL-4B + DiT 动作专家） |
| 动作空间 | 50×8（7 关节 + 1 夹爪），单臂 |
| 已验证 | Phase 1 本地 4-bit 推理跑通（RTX 4060, 8GB VRAM） |
| 输入模态 | 3 路 RGB 224×224 + 文本指令 + 关节状态 |
| 触觉编码 | 规划中（12×12 网格 + 3 通道 + VQ-VAE+CNN） |

### 能力分层评估

#### ✅ 可以覆盖（约 40%）

| 操作 | 理由 |
|------|------|
| 场景理解（识别样品/杯/池） | Qwen3-VL 视觉理解强项 |
| 粗加样、粗抓取 | DiT DiT 开环生成 ±1-3mm 够用 |
| 接气路、取池取杯 | 类似 pick-and-place，Phase 2 已验证 |
| 清洁、数据记录 | 粗动作 + 逻辑 |

#### ⚠️ 需改造才能覆盖（约 40%）

| 操作 | 缺口 | 解决路径 |
|------|------|---------|
| 抹平（0.05-0.10N） | 无力反馈 | 触觉编码融合（Phase 3） |
| 精密装配（O形圈/窗片） | ±0.15mm 精度 | 视觉伺服层 + AprilTag |
| 杯位调整 | 微米级对光 | 专用近景相机闭环 |
| 温程触发采谱 | 非动作输出 | 任务规划器扩展 |

#### ❌ 当前做不到（约 20%）

| 需求 | 原因 | 方案 |
|------|------|------|
| 双臂协同 | 动作空间 8 维（单臂） | 智元机器人双臂由本体 SDK 处理，LabVLA 输出 16 维动作空间 |
| 工具切换 | 无工具库管理 | 上层任务规划器调度 |
| 24 步编排 | 无状态机 | LLM 任务规划层 |
| 在线力反馈 | 开环生成无闭环 | 触觉→DiT 在线修正环路 |

## 三、推荐架构方案

### 四层互补

```
┌──────────────────────────────────────────────┐
│  ① 任务规划层 (LLM)                           │
│  编排 24 步状态机、异常处理、分支决策           │
│  ± 不涉及精度，依赖 LLM 推理能力               │
├──────────────────────────────────────────────┤
│  ② LabVLA 动作生成 (DiT)                     │
│  场景理解 + 粗轨迹生成，±1-3mm                │
│  3 路 RGB 输入，输出 50 步动作 chunk          │
├──────────────────────────────────────────────┤
│  ③ 视觉伺服精对准 (手眼相机 + AprilTag)        │
│  手腕微距相机 + 工位基准标记，±0.1mm          │
│  PID 闭环微调末端位置                          │
├──────────────────────────────────────────────┤
│  ④ 触觉力控闭环 (124 点手套)                  │
│  抹平/装配时的 0.05-0.10N 接触检测             │
│  VQ-VAE 编码 → 触觉特征 → DiT 在线修正        │
└──────────────────────────────────────────────┘
```

### 传感器精度分级

| 层级 | 传感器 | 精度 | 频率 | 用途 |
|------|--------|------|------|------|
| 顶层 | LabVLA 3路 RGB | ±1-3mm | ~10Hz | 场景理解、粗轨迹 |
| 中层 | 俯视+手腕相机+AprilTag | ±0.1mm | 30-60Hz | 精对准、工具定位 |
| 底层 | 触觉手套 124点 | 0.05N | ~100Hz | 接触力闭环 |

### 相机选型建议

| 相机 | 视场 | 像素尺度 | 用途 |
|------|------|---------|------|
| 腕部 RGBD（机器人自带） | 大视场 | 0.136mm/px | 发现、抓取 |
| 顶架俯瞰 RealSense D435 | ~1m | 0.1mm/px | 全局定位 |
| 手腕微距相机 | 50mm | **0.026mm/px** | 杯口精对准 ⭐ |

## 四、与触觉手套项目的衔接

### 抹平操作 — 触觉手套的杀手级用例

抹平步骤要求 0.05-0.10N 触碰精度，正好落在柔性织物触觉传感器（124 点）的能力区间。

**两种使用模式**：

| 模式 | 用途 | 工作方式 |
|------|------|---------|
| **示教器** | 训练数据采集 | 人手戴手套做抹平 → 记录 124 点压力分布 → 作为机器人训练数据 |
| **在线传感器** | 实时力反馈 | 机械臂指尖贴传感器 → 检测接触力 → 修正抹平轨迹 |

### 触发编码融合路径

```
触觉 124 点 → reshape 12×12 网格 → 3 通道
  (Ch1 压力 / Ch2 变化率 / Ch3 振动)
  → CNN 编码 → 64 维特征 → 接入 LabVLA DiT
  （走 A+B 混合：视觉层 + 状态层都加触觉）
```
