## 🔄 同步方式

使用 **Obsidian Git** 插件自动同步，跨多台电脑共享。

### 首次使用

```bash
git clone git@github.com:majosan/josan-obsidian-vault.git
```

在 Obsidian 中打开该文件夹作为 vault，然后：
1. 设置 → 第三方插件 → 安全模式关闭
2. 社区插件 → 搜索并安装 **Obsidian Git**
3. 启用后设置：
   - Auto commit & push：5 分钟
   - Auto pull：10 分钟
   - Pull on startup：开启
