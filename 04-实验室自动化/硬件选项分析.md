## 三、框架/模型路线（软件栈）

```
应用层 → 材料催化特定动作
  ↑
模型层 → LABVLA (Intern-VLA) / OpenVLA / Octo
  ↑
训练层 → Intern-Robotics 训测引擎 / Isaac Lab / Robomimic
  ↑
仿真层 → Intern-Utopia 仿真引擎 / Isaac Sim
  ↑
中间层 → ROS2 + 你自研的数据采集管道
  ↑
硬件层 → 选定臂 + 灵巧手 + 你的数据手套
```

### 框架对比

| 框架 | 厂家 | 开源 | 特点 |
|------|------|------|------|
| **Intern-Robotics** ⭐ | 上海AI Lab | ✅ 全栈开源 | 仿真+数据+训测一体，支持轮式，LABVLA兼容 |
| **LABVLA** 🎯 | 上海AI Lab | ✅ 已部署 | Intern-VLA系列，你已跑通Phase1/2 |
| **OpenVLA** | Stanford/Berkeley | ✅ 开源 | 7B VLA模型，行为克隆 |
| **Isaac Lab** | NVIDIA | ✅ 开源 | 最强仿真，配MimicGen自动扩数据 |
| **Robomimic** | MIT | ✅ 开源 | 最成熟模仿学习框架 |
| **Octo** | UC Berkeley | ✅ 开源 | 多机器人基座模型 |
| **RDT** | 清华 | ✅ 开源 | 灵巧操作基座模型 |
