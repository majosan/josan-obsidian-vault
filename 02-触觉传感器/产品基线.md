## 二、核心技术方案

### 2.1 传感器结构（三层织物）

```
┌──────────────────────────────────┐
│  上层：32根镀银纱线（横向）        │
│  刺绣锯齿形走线                    │
├──────────────────────────────────┤
│  中层：压敏纱线（尼龙+碳黑涂层）    │
├──────────────────────────────────┤
│  下层：32根镀银纱线（纵向）        │
│  刺绣锯齿形走线                    │
└──────────────────────────────────┘
```

- 上下层正交排列 → 交叉点 = 传感单元
- 全织物结构，无刚性元件

### 2.2 两个 SKU 的差异化设计

| 参数 | SKU-1（小） | SKU-2（大） |
|------|:---------:|:---------:|
| 整体尺寸 | 500mm × 500mm | 1400mm × 700mm |
| 阵列 | 32 × 32 | 32 × 32 |
| 锯齿形齿高 | 3mm | **4mm** |
| 传感点尺寸 | 3mm × 3mm | **4mm × 4mm** |
| 行间距（纵向pitch） | 13mm | **35mm** |
| 列间距（横向pitch） | 13mm | **17mm** |
| 有效传感区域 | ~403mm × 403mm | ~1085mm × 527mm |
| 网格形状 | 正方形 | **矩形（行疏列密）** |

> SKU-2 的网格是矩形分布的：纵向间距大（35mm）、横向间距小（17mm），适合覆盖大面积但不需要均匀密度的场景。

### 2.3 传感原理

```
压力 → 压敏纱线形变 → 接触电阻↓ → 行列扫描 → 12-bit ADC → JSON输出
```

## 三、产品规格（已确认）

### 3.1 物理参数

| 参数 | SKU-1 | SKU-2 |
|------|:-----:|:-----:|
| 阵列密度 | 32×32 = 1024点 | 32×32 = 1024点 |
| 传感点尺寸 | 3×3mm | 4×4mm |

| 行间距 | 13mm | 35mm |
| 有效传感区域 | 403×403mm | 1085×527mm |
| 电极工艺 | 镀银纱线，刺绣锯齿形 | 同 |
| 压敏材料 | 尼龙基底 + 碳黑涂层 | 同 |

### 3.2 电学参数

| 参数 | 数值 |
|------|------|
| ADC 精度 | **12-bit**（4096级），软件映射至 0-255 |
| 采集帧率 | **15 Hz** |
| 输出接口 | 串口 → USB（单线供电+通讯） |
| 供电 | 5V USB |
| 数据格式 | JSON |
| 最小可检测压力 | 约 **5g**（砝码实测） |

### 我们的差异化

- 全织物透气可折叠 vs 薄膜式不透气
- 大角度弯折甚至折叠 vs 薄膜受限
- 3-4mm 微小传感点
- 电路隔离好 → 低串扰
