- "十四五"规划中明确传感器为关键核心器件

**具体利好政策**

5. **医疗器械国产化**：康复设备进口替代导向

## 2. 各行业中国市场特殊分析

### 2.1 科研教育（中国市场焦点）

**市场特征**
- 中国高校数量：~3,000所，其中理工科院校 ~1,200所
- 设有机械/机器人/生物力学相关实验室：~500所
- 重点目标：985工程大学（39所）+ 211工程大学（112所）+ 中科院系统
- 科研仪器采购预算：单校每年 **数百万至数千万** 用于实验设备

**采购特点**
| 特征 | 说明 |
|------|------|
| 招标为主 | 多数高校必须通过招标采购（金额>10万） |
| 国产优先 | 政府对国产仪器有明确倾向政策 |
| 论文驱动 | 能出论文的设备最好卖 |
| 预算集中 | 3-4月和9-10月为采购旺季 |
| 导师话语权 | 课题组导师决定采购意向 |

**机会分析**
- **定价策略**：对标Tekscan的1/3-1/5 → ~¥10,000-¥50,000/套
- **论文合作**：提供免费样机给重点实验室发表论文
- **教学套件**：¥2,000-¥5,000的学生实验套件
- **国产标签**：直接竞争力（"国产替代Tekscan"）

**数据来源：** 教育部高校统计数据、国家科技报告服务系统
**置信度：** 中高

### 2.2 智能制造/工业检测（中国市场焦点）

**市场特征**
- 中国制造业增加值全球第一（~35%份额）
- 工业传感器市场：约 **1,500亿人民币**（含所有类型）
- 工业压力分布检测细分：约 **8-12亿人民币**
- 锂电池制造产线检测是增长最快的方向

**锂电池制造细分**
| 环节 | 检测需求 | 传感器要求 |
|------|---------|-----------|
| 电芯堆叠 | 压力均匀性 | 大面积阵列 |
| 卷绕 | 张力均匀性 | 狭长形状 |
| 极片涂布 | 面密度 | 高精度 |
| 化成过程 | 膨胀压力 | 长期监测 |

**OEM/ODM机会**
- 大部分中国工业客户**不愿购买完整系统**，而需要**定制集成方案**
- 企业更愿意买传感器硬件+SDK，自行集成到产线
- 与服务商合作模式：提供传感模组，收取开发费+出货提成

**数据来源：** 中国工信部智能制造装备创新报告、锂电池行业协会
**置信度：** 中

### 2.3 医疗康复（中国特殊机会）

**市场特征**
| 因素 | 说明 |
|------|------|
| 老龄化 | 60岁以上人口3亿+，康复需求爆发 |
| 政策 | 康复医学纳入二级及以上医院基本建设要求 |
| 支付 | 多地医保已将康复项目纳入报销 |
| 家用化 | 康复设备从医院向家庭转移趋势 |

**重点应用场景**
1. **足底压力分析** — 步态评估、糖尿病足预防
2. **压力疮预防** — 老年护理院、ICU、居家护理
3. **假肢适配** — 接受腔压力分布检测
4. **康复评估** — 平衡训练、坐姿压力分析

**监管壁垒**
- 二类医疗器械注册需要 **12-24个月**
- 费用：~¥30万-¥100万（含临床试验）
- 但：研发阶段的非医疗设备无需注册

**策略建议**
- 先以"科研工具"进入（需注册证），积累临床数据
- 待产品稳定性验证后，再申请医疗器械注册证
- 优先选择与医院合作研发（可走科研项目通道，豁免部分注册要求）

**数据来源：** 国家药监局(NMPA)相关规定、卫健委统计数据
**置信度：** 中高

| 技术路线 | 代表企业 | 数量 | 量产进展 |

| **压阻式（织物基）** | **本产品**、矩侨工业、尧乐科技 | 3家 | 均Demo/小批量阶段 |
| **电容式** | 他山科技、赛感科技、申昊科技、韧和科技、哈普泰克、猿声科技 | ~6家 | 他山芯片已供货 |

| **电磁感应式** | 越疆科技 | 1家 | 自用（SafeSkin） |

**关键发现：**
- **织物压阻路线仅3家企业，是最小众的方向** — 差异化优势明显，但也意味着未被充分验证
- 压阻印刷/涂覆路线最拥挤（~15家），且已有规模量产案例
- 各路线尚无明确胜出者，技术路线之战仍在持续

```

     ↓ 量产追赶

```

**关键判断（更新版）**：
- 中国电子皮肤市场正从"进口替代"进入"国产企业混战"阶段
- **中国已有量产案例（华威科2000只）**，不再是纯概念验证
- 国产企业的竞争将**在未来1-2年加剧**，窗口期缩短
- **全织物路线仅3家企业**，是最大的差异化空白
- 建议：**尽快完成小批量试产，锁定2-3家标杆客户**
